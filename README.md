# Svika

**Digital tickets, real revenue, same kombi.**

A digital ticketing and trip-planning system for Harare's informal kombi network. Submitted to the GDG Harare Build with AI Hackathon 2026.

## What's working — Tier 1

_Not yet built. Phase 0 in progress._ Once shipped:
- Trip planning across four routes with three viable trips
- Transferable digital tickets
- Conductor PIN flow plus cash walk-on
- Live kombi positions on a map (simulated)
- Fleet dashboard with per-kombi revenue ledger
- Bilingual Ghost Trip audit narrative generated by Gemma
- ZIMRA liability estimate
- WhatsApp-style companion with three commands
- Persona switching via `?as=` query parameter

## What's demonstrated — Tier 2

- Same-kombi parcel happy path (single PIN handover)
- Optional: emergency-contacts card on the dashboard
- Optional: feature-phone menu mock previewing the carrier-menu roadmap

## Roadmap — Tier 3

See [`docs/ROADMAP.md`](docs/ROADMAP.md) for the full list. Highlights:
- Carrier menu access (`*123#`)

## Stack

- Next.js 16 · React 19.2 · TypeScript 5
- Tailwind CSS v4 (CSS-first config)
- Supabase Postgres + PostGIS + Realtime
- Mapbox GL JS v3
- Gemma 4 E2B via Ollama (local) · Gemini 2.5 Flash (fallback)
- Vercel Hobby

## How to run

```bash
pnpm install
cp .env.example .env.local           # then fill in real keys
ollama pull gemma4:e2b-it-q4_K_M     # ~7.2 GB
pnpm ai:spike                        # Phase 0 verification
supabase login && supabase link --project-ref YOUR-REF
pnpm db:migrate
pnpm db:seed
pnpm dev
```

## Demo personas

The user-facing flow is **single-user**. Takunda is the only persona the landing page surfaces. The other URLs exist for direct deep-links (used during demo recording) but are not offered as switches in the UI.

- `/?as=takunda` — primary user (passenger). The landing page funnels everyone here.
- `/?as=rudo` — transfer recipient (cousin); reachable through a shared claim link.
- `/hwindi?as=farai` — conductor character on `ZH 4821`. Direct URL only.
- `/fleet?as=baba_tino` — fleet owner character. Direct URL only.
- `/wa?as=takunda` — WhatsApp companion (mocked).
