# video-assets/

Source materials for the Svika hackathon submission video.

## Structure

- references/
  - brand/      Logo, wordmark, palette, typography, brand HTML system
  - mockups/    V1 app screenshots, 8 frames in scene order
  - context/    Reference photos and source artwork
- audio/        Voiceover and music (added during production)
- generated/    AI-tool output clips
- final/        Final rendered MP4

## How to use

Any AI video tool that ingests text and images can be pointed at this folder. The CONTEXT.md file at the top is the single source of truth for what Svika is and what the video has to communicate. SCRIPT.md is the locked voice script.
